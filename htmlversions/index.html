HTML 1.0

HTML 1.0 was created in 1991 by Tim Berners-Lee, it was the first standardized version of HTML.
HTML 1.0 wasn't very functional as it mostly focused on structuring and displaying text documents.
Basic elements like headings (h1-h6), paragraphs (P tags), lists, and links were introduced.

HTML 2.0

HTML 2.0 was introduced in 1995, it was first known as RFC 1866. HTML 2.0 introduced support for tables, allowing better organization of data.
The concept of forms was introduced as well, form tags were used to enable interactive web functionslike search queries and login forms.

HTML 3.2

HTML 3.2,(introduced in 1997), enhanced support for tables, forms, and text formatting.
HTML 3.2 also introduced new elements like image maps ( allow users to interact with images that include multiple clickable areas called hotspots), applets (small programs written in java that could be embedded in HTML using the (applet) tag), and basic support for style sheets.

HTML 4.01

In 1999 HTML 4.01 was introduced, key features included improved support for style sheets, scripting, frames, and tables.
This also included strong focus on making web pages more acceseble through features like improved forms, accesskey atricutes and detailed markup fo robjects.
